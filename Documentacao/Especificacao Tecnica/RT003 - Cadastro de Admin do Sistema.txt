RT001 - Envio de Email
RT002 - Cadastro de Usuario

Usuários Admin são usuários do sistema que irao adminsitrar a plataforma. Nesse caso, é cadastrado um usuário e esse é inserido no perfil de Admin. Apos o cadatro, é enviado um email com o texto:
   Voce foi cadastrado na Plataforma Zumira no perfil de Admin com a senha xxxx. Efetue o login no sistema pelo link abaixo e efetue a troca da sua senha.

Link do sistema:
wwww.xpto.com.br

Apos o envio do email, deve ser gravado o log  para esse novo usuário com a seguinte descrição:
E-mail notificacao novo usuário gerado, Id xxxx, sendo xxxx o Id da msg de email retornado pelo micro-servico de envio de emails
